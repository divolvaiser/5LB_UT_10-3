Справочник "Учетные записи электронного обмена"

Справочник предназначен для использования в механизме обмена электронными документами между различными информационными базами и содержит настройки этого обмена.

  • Владелец (Организация) - по которой создана учетная запись
  • Идентификатор - Уникальный идентификатор, однозначно идентифицирующий организацию в информационных базах клиентов.
  • Способ отправки - способ отправки (получения) электронных документов.
    Возможные способы отправки:
    • Отправлять на сервер
    • Выгружать в файл
    • Отправлять по электронной почте
  • Время ожидания ответа время ожидания ответа от контрагента по умолчанию.

Параметры, необходимые для отправки электронных документов ч-з сервер:

  • Имя сервера
  • Ресурс сервера
  • Порт сервера
  • Имя пользователя на сервере
  • Пароль пользователя на сервере

 

Параметры, необходимые для отправки электронных документов ч-з файл:

  • Каталог для выгрузки файлов
  • Каталог для загрузки файлов

Параметры, необходимые для отправки электронных документов по электронной почте:

  • Учетная запись электронной почты

 

На ряду с основными настройками для учетной записи необходимо определить состав контрагентов, их идентификаторы и адреса электронной почты (для способа отправки "по электронной почте")
