Справочник «Виды распределения по проектам»

В справочнике задается наименование вида распределения по проектам, в том случае, если сумма одной номенклатурной позиции или сумма по одному документу должна распределяться по нескольким проектам. Доли распределения по проектам задаются документом "Установка долей распределения по проектам".

Вид распределения по проекту для позиции номенклатуры задается в справочнике "Номенклатура" на закладке "Проекты".  Вид распределения в целом по документу для документов поступления или отгрузки задается на закладке "Дополнительно". Для документов движения денежных средств вид распределения по проектам  задается в форме документа в соответствующем поле.

При проведении документов поступления и отгрузки используется следующая стратегия: если проект или вид распределения по проектам не задан в целом по документу, то тогда используются те проекты или виды распределения по проектам, которые заданы для позиций номенклатуры, указанных в табличной части документа.  

Важно! Использование видов распределения по проектам возможно только в том случае, если в настройках параметров учета установлен флаг "Использовать виды распределения по проектам".
