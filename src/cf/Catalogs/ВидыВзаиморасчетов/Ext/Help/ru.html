Справочник "Виды взаиморасчетов"

Справочник предназначен для дополнительного разделения взаиморасчетов для анализа данных в отчетах.

С помощью этого справочника можно произвести дополнительную классификацию однотипных договоров различных контрагентов.

Например, для  долгосрочных договоров установить вид взаиморасчетов - долгосрочные, а для разовых договоров установить вид взаиморасчетов - разовые.
