Задачи с оповещением. Форма списка задач

Общие сведения

Список задач вызывается из пункта глвного меню системы: «Сервис»-«Задачи с оповещениями»-«Открыть список», и представляет собой форму, в основном списке которой перечисляются задачи, для которых текущий пользователь определен как автор либо исполнитель. Вызов формы можно осуществлять при запуске системы автоматически,- для этой цели, в нижней своей части, форма снабжена флагом "Отображать форму при запуске системы".

Действия списка

Задачи к исполнению (в которых данный пользователь является исполнителем) и задачи для контроля прочих исполнителей (в которых данный пользователь является автором), отображаются в форме отдельными списками, исходя из выбранного режима отображения, переключение которого производится с помощью кнопки на панели управления с изображением:  . Необходимо помнить, что в оба списка задача попадет только в том случае, если Вы являетесь её автором и исполнителем одновременно (задача, поставленная самому себе).

При первом открытии в списке показываются только активные задачи пользователя. Пользуясь кнопкой панели управления с изображением: , форма включает либо отключает отображения только активных, либо соответственно всех (включая завершенные) задач текущего пользователя.

Для дополнительного удобства, форма снабжена полем, отображающим детали выбранной в списке задачи. По умолчанию, поле деталей скрыто, но может быть отображено в нижней части формы в результате нажатия кнопки с изображением: .

Создание новой задачи

Форма списка позволяет создавать новые задачи. Для этой цели предусмотрен стандартный пункт "Добавить", меню "Действия", командной панели. Аналогично созданию новой задачи путем добавления, форма предоставляет возможность создания новой задачи путем копирования существующей. Редактирование значений параметров новой задачи производится в полях объекта "Форма задачи".

Действия над задачами

Существующие средства формы списка позволяют производить открытие объекта "Форма задачи", двойным щелчком по позиции списка, а также, не открывая задачу в отдельной форме, помечать её как выполненную. Данная возможность предоставляется благодаря наличию в командной панели кнопки с изображением: .
