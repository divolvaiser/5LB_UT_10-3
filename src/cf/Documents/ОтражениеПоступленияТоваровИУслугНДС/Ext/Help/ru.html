Документ "Отражение поступления товаров и услуг (НДС)"

Документ "Отражение поступления товаров и услуг (НДС)" предназначен для отражения поступления товаров и услуг в подсистеме учета НДС при условии, что учет поступления ведется вручную.

В реквизите формы "Документ расчетов" выбирается документ, отразивший поступление в бухгалтерском учете. При выборе пункта подменю "Заполнить", табличная часть автоматически заполняется на основании документа расчетов.

Для отражения поступления, как товаров, так и услуг используется одна табличная часть.

При проведении документа формируются соответствующие записи в регистрах подсистемы учета НДС.

Документ также используется при вводе начальных остатков по НДС. В этом случае он автоматически создается при записи документа "Ввод начальных остатков НДС" (с видом операции "НДС по приобретенным ценностям") с комментарием вида "Создан автоматически документом <Ввод начальных остатков НДС ...>". При этом недоступно редактирование реквизитов документа.

Для документа можно ввести счет-фактуру полученный.
