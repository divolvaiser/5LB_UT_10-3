Документ "Электронное письмо"

Документ предназначен для отправки исходящих электронных писем и регистрации входящих в подсистеме "Встроенный почтовый клиент".

Использование данного документа возможно только при включенном встроенном почтовом клиенте в настройках учета.

Каждое электронное письма может находиться в одном из состояний:

  • Полученное

  • Отправленное

  • Исходящее

  • Сохраненное

Данное состояние изменяется пользователем косвенно, при совершении тех или иных действий (создание и сохранение письма, отправка, получение).

Все входящие письма являются полученными. Статус «Сохраненное» письмо принимает, когда является так называемым «черновиком».

Исходящее письмо – письмо к отправке, но по тем или иным причинам еще не отправленное на почтовый сервер (например у пользователя нет прав на транспорт писем, нет связи с почтовым сервером). В состоянии «Исходящее» письмо еще может быть отредактировано.

Все полученные письма недоступны для редактирования, кроме тех полей, которые не характеризуют электронное письмо как почтовое сообщение, а являются некими классификаторами данного письма в текущей конфигурации (например группа писем, предмет и пр.).

Для нового исходящего письма можно указать тип текста письма (простой текст, текст в формате HTML). Для текста в формате HTML в тело письма можно вставлять картинки.

В полях «Кому», «Копии» и «Скрытые копии» пользователь может указать начало имени или адреса электронной почты адресата. Программа сама пытается отыскать по данным параметрам получателя. При положительном и единственном результате поиска строка получателя автоматически преобразуется к виду:

Имя получателя <Адрес электронной почты получателя>.

Имя получателя может отсутствовать, тогда отображается только адрес электронной почты получателя (например, если адрес электронной почты введен вручную пользователем).

Если с введенными параметрами найдено несколько получателей, то при окончании редактирования пользователь имеет возможность выбрать получателя из найденного списка.

Несколько адресов или имен получателей можно указывать через запятую. В выпадающих списках выбора данных полей можно найти значения последних получателей.

По кнопке выбора в полях "Кому", "Копии" и "Скрытые копии" открывается адресная книга для подбора получателей письма.

На закладке "Дополнительно" можно указать группу писем учетной записи, которой данное письмо принадлежит, предмет данного письма.

На закладке вложения производятся операции добавления, просмотра и удаления вложений письма.

В подменю "Действия" командной панели формы по кнопке "Ввести событие" можно на основании электронного письма создать новый документ "Событие", по кнопке "Скрытые копии" можно управлять видимостью поля формы "Скрытые копии".
