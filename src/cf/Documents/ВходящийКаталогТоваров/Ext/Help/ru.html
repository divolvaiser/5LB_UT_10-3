Документ "Входящий каталог товаров"

Документ не создается интерактивно. Создание документа происходит автоматически (при нажатии на кнопку "Получить" в журнале документов Электронные документы), когда контрагент со своей стороны отправляет электронный документ Исходящий каталог товаров. 

После получения электронного документа "Входящий каталог товаров" необходимо выполнить загрузку всех или некоторых строк каталога.

Загрузка строк каталога производится из пункта меню "Действия" командной панели "Каталог товаров", нажатием на пункты "Загрузить выбранные строки каталога", "Загрузить все строки каталога".

При нажатии на эти кнопки открывается форма "Настройки загрузки каталога" на которой необходимо определить следующие настройки:

  • Группа номенклатуры для новых элементов
  • Перемещать существующие элементы в группу
  • Обновлять свойства существующим элементам
  • Обновлять штрих коды существующим элементам
  • Не загружать, если штрих код установлен для другой номенклатуры

В случае, если аналогичная строка каталога уже существует в справочнике "Номенклатура", то необходимо установить соответствие, выбрав нужный номенклатурный элемент в колонке "Номенклатура". Для строк каталога, по которым ранее производилась загрузка, значение в колонке "Номенклатура" устанавливается автоматически.

При нажатии на кнопку "Принять к работе" создается документ Исходящий товар к работе и устанавливается флаг "Обработан".
