Документ "Требование-накладная"

Документ предназначен для оформления операции передачи товаров для внутреннего использования и списания факта использования товаров для внутреннего использования по определенной статье затрат.

Документ может быть оформлен на основании документа «Внутренний заказ». В документе "Внутренний заказ" должен быть установлен вид заказчика - «В подразделение».

Документ "Требование-накладная" может быть оформлен по нескольким внутренним заказам. В этом случае внутренние заказы указываются в табличной части документа. 

В табличной части документа может быть указан и документ «Заказ покупателя». Такая операция оформляется в том случае, если какой-то товар передается покупателю безвозмездно, например, в качестве дополнительного подарка. Причем в одном документе "Требование-накладная" можно указать несколько заказов покупателей.  

В одном документе «Требование-накладная» можно отразить списание товаров на любую из статей затрат различного вида: общехозяйственные (ОХР), коммерческие (КР), вложения во внеоборотные активы, прочие. То есть по каждой позиции номенклатуры можно указать свою статью затрат. Список статей затрат хранится в справочнике «Статьи затрат» и может произвольно дополняться пользователем.

Номенклатурная группа указывается для дополнительной аналитики затрат, то есть для отнесения затрат на определенную номенклатурную группу товаров для дальнейшего распределения по ней косвенных расходов при передаче данных в бухгалтерию.
