Механизм обмена данными. Обмен данными Управление торговлей - Бухгалтерия.

 

План обмена предназначен для хранения изменений и фильтрации данных при обмене Управление торговлей - Бухгалтерия.

Имеется возможность фильтровать данные при обмене по организациям, складам и подразделениям, по принадлежности документов к бухгалтерскому учету. Отдельная настройка позволяет настроить перенос себестоимости списания. Можно настроить выгрузку аналитики по складам.

Префикс загружаемых документов определяет добавляемую часть к номеру объектов, которые загружаются при обмене. Префикс добавляется только для документов, дата которых не меньше чем начальная дата обмена документами. Начальная дата так же определяет фильтрацию выгружаемых документов. Выгрузке подлежат только те документы, дата которых не меньше указанной.
