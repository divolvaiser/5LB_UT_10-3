Отчет по наличию счетов-фактур

Данный отчет предназначен для получения информации о наличии счетов-фактур, сформированных на основании документов, заданных в настройках отчета.

Счета-фактуры на выданные авансы отражаются в отчете только при наличии проведенных документов "Счет-фактура полученный". Платежные документы, по которым нет счетов-фактур, в отчете не выводятся.

Настройки отчета

Настройки отчета устанавливаются в главной форме отчета. Можно задать следующие параметры отчета:

  • Даты начала и окончания отчетного периода.
  • Организация. Наименование организации, для бухгалтерского учета которой формируется  отчет.
  • Наличие счета-фактуры. Признак включения в отчет информации о наличии счетов-фактур, сформированных на основании указанных документов. Возможные значения: «Есть» (при построении отчета учитываются только документы, по которым сформированы соответствующие счета-фактуры), «Нет» (учитываются только документы, по которым не были сформированы счета-фактуры), «Не имеет значения» (факт формирования счетов-фактур по указанным документам не принимается во внимание).

В табличной части блока «Список документов» выбираются необходимые документы. В зависимости от значения параметра «Отбор» отчет формируется либо на основании документов, указанных в табличной части блока «Список документов», либо на основании всех документов, кроме указанных в табличной части блока «Список документов». Если список документов не заполнен, отчет формируется по всем документам с учетом остальных ограничений (организация, период и т.п.).

Результаты формирования отчета помещается в блок «Результат» и представляет собой таблицу со следующими полями:

  • Документ-основание. Наименование документа.
  • Счет-фактура. Наименование счета-фактуры, сформированного на основании данного документа.
  • Проведен. Признак проведения счета-фактуры.
