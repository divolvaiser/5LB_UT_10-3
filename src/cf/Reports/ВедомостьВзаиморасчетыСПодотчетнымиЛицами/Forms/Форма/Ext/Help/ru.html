Отчет «Ведомость по взаиморасчетам с подотчетными лицами»

В отчете «Ведомость по взаиморасчетам с подотчетными лицами» можно получить полную информацию о взаиморасчетах с подотчетными лицами.

Для детализации взаиморасчетов с подотчетными лицами по документам в списке группировок отчета должна присутствовать группировка «Документ движения (регистратор)».

В отчете будет отображаться документ «Расходный кассовый ордер», по которому подотчетному лицу были выданы денежные средства и документ «Авансовый отчет», по которому подотчетное лицо отчитывалось за выданные ему средства.

В том случае, если подотчетное лицо возвращало неистраченные денежные средства, в отчете будет отображен документ «Приходный кассовый ордер», по которому оформлялся возврат денежных средств от подотчетного лица.

Если установить группировку "Расчетный документ"- "Документ движения (регистратор)", то данные в отчете будут сгруппированы по расчетным документам. В качестве расчетного документа может быть документ "Расходный кассовый ордер" или "Авансовый отчет". При указании такой группировки можно получить информацию о том, как подотчетное лицо отчиталось за выданные ему денежные средства. То есть можно отследить взаимосвязь между документами "Расходный кассовый ордер", которым были выданы денежные средства подотчетному лицу,  документом "Авансовый отчет", с помощью которого подотчетное лицо отчиталось за выданные ему средства и документом "Приходный кассовый ордер", по которому оформлялся возврат денежных средств подотчетному лицу.

Информация в отчете выводится в двух валютах: валюте управленческого учета («Сумма в USD») и валюте взаиморасчетов с подотчетным лицом («Сумма в валюте взаиморасчетов»).

Валюта взаиморасчетов с подотчетным лицом назначается в документе «Расходный кассовый ордер» при выдаче денежных средств подотчетному лицу.

С помощью установки значений отбора, представленных в отчете, можно отобрать информацию по конкретному подотчетному лицу, группе подотчетных лиц, произвольному списку подотчетных лиц, по подотчетным лицам, имеющим определенные свойства и категории.
