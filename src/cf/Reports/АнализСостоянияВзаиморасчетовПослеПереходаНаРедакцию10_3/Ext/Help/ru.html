Отчет «Анализ состояния взаиморасчетов после перехода на редакцию 10.3»

После выполнения обновления данных необходимо провести сверку данные по регистрам Взаиморасчеты с контрагентами и Взаиморасчеты с контрагентами по документам расчетов, чтобы убедиться в корректности проведенного обновления.

Для сверки используется специальный отчет Анализ состояния взаиморасчетов после перехода на редакцию 10.3. В отчет выводятся следующие данные:

  • Остатки и обороты по регистру Взаиморасчеты с контрагентами для договоров, по которым взаиморасчеты ведутся по документам расчетов. Документ расчетов определяется на основании измерения Сделка.
  • Остатки и обороты по регистру Взаиморасчеты с контрагентами по документам расчетов. Документ расчетов определяется на основании измерения Документ расчетов с контрагентом
  • Расхождения между остатками и оборотами по этим двум регистрам

Состав группировок отчета:

  • Организация
  • Контрагент
  • Договор контрагента
  • Сделка / Документ расчетов с контрагентом.

При необходимости в состав группировок может быть добавлен регистратор и период.

 

После того, как будут сверены взаиморасчеты, и будет достоверно известно что сведения в регистре Взаиморасчеты с контрагентами по документам расчетов заполнились корректно, необходимо удалить устаревшие данные из документов и регистров. Для этих целей также используется отчет Анализ состояния взаиморасчетов после перехода на редакцию 10.3

  • В документах, в которых используется договор с установленным признаком Вести по документам расчетов с контрагентом, необходимо очистить реквизит Сделка. Для этого необходимо нажать кнопку Дополнительно – Очистить реквизит Сделка в документах.
  • В движениях регистров
    • Взаиморасчеты с контрагентами,
    • Расчеты с контрагентами
    • Движения денежных средств
    • Заявки на расходование средств
    • Планируемые поступления денежных средств
    • Реализованные товары
    • Товары переданные
    • Товары полученные
    • Денежные средства комиссионера
    • Денежные средства комитента
      в которых используется договор с установленным признаком Вести по документам расчетов с контрагентом, необходимо очистить значение измерения Сделка. Для этого необходимо нажать кнопку Дополнительно – Очистить измерение Сделка в регистрах.
  • В договорах с контрагентами, в которых способ ведения взаиморасчетов в ред.10.2 был «По документам расчетов», нужно заменить его на «По договору в целом». Для этого необходимо нажать кнопку Дополнительно – Изменить ведение взаиморасчетов по договорам.

 

После того, как будут выполнены перечисленные выше действия по удалению устаревших данных, отчет Анализ состояния взаиморасчетов после перехода на редакцию 10.3 уже не сможет сопоставить взаиморасчеты в разрезе документов расчетов по разным регистрам взаиморасчетов – в регистре Взаиморасчеты с контрагентами для этого уже не будет требуемых сведений.
