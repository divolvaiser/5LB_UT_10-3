Отчет «Планирование закупок»

Отчет предназначен для количественного планирования закупок в разрезе складов на основании данных управленческого учета о продажах.

Закладка «Параметры»

На закладке устанавливаются основные параметры, влияющие на расчет планируемых показателей.

  • Период анализа продаж - фактический период, на основании которого рассчитываются средние продажи.
  • Период закупок - период, на который планируется проводить закупки.
  • Флажок «Показать только необходимые закупки» - в отчет выведутся только те номенклатурные позиции, по которым необходимый объем закупки больше нуля.
  • Флажок «По всей номенклатуре» - отчет сформируется по всем позициям из справочника «Номенклатура».
  • Тип цен номенклатуры - заполняется, если отчет будет использоваться для формирования и заполнения заказов поставщикам.
  • Рабочая неделя - параметр, необходимый для расчета числа рабочих дней в планируемом периоде.
  • Остаток к началу периода закупок рассчитывается по двум вариантам. Получить из итогов - исходя из данных по остаткам на складах; Рассчитать с учетом заказов - данные по остаткам на складах будут увеличены на объем ожидаемых поступлений согласно оформленным заказам поставщикам.

Закладка «Отбор»

Здесь можно ограничить отбор данных для отчета в разрезе складов, номенклатуры и ее характеристик.

Закладка «Дополнительные поля»

Закладка позволяет добавить в отчет необходимые колонки с характеристиками номенклатуры.

Закладка «Показатели»

На закладке выбираются показатели, которые необходимо вывести в отчет.

  • Продано - количество реализованного товара за анализируемый период; определяется в единицах хранения с учетом возвратов.
  • Дней продажи - количество дней, когда товар был на складе и/или продавался.
  • Средняя продажа за день - определяется делением объема продаж на количество дней продажи.
  • Остаток на дату начала закупок - остаток на складе.
  • Необходимый объем закупок - определяется путем вычитания остатка на складах из планируемого объема продаж.
  • Планируемый объем продаж - произведение средней продажи за день на количество рабочих дней в планируемом периоде.

На основании отчета можно оперативно сформировать документы «Заказ поставщику» и «Внутренний заказ» в разрезе складов.
