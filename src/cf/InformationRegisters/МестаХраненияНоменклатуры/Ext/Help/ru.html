Регистр сведений "Места хранения номенклатуры"

В регистре сведений "Места хранения номенклатуры" хранится информация о распределении позиций номенклатуры по местам хранения.

Перед тем , как заполнять информацию о местах хранения номенклатуры, в справочнике "Места хранения" должна быть задана структура мест хранения по каждому складу.

Информация о расположении товаров по местам хранения используется при печати документов (форма "Бланк товарного наполнения").

Сведения о местах хранения номенклатуры заполняются по каждому складу.  Для заполнения сведений о местах хранения номенклатуры, необходимо в справочнике "Склады" установить курсор на нужный склад, нажать на кнопку "Перейти" и выбрать пункт меню "Места хранения номенклатуры".  

Если товар хранится на одном складе в разных местах хранения, то в регистре сведений "Места хранения номенклатуры" в поле "Приоритет" необходимо заполнить приоритет для этих мест хранения.

По основному месту хранения приоритет заполнять не надо (нулевой приоритет). Для дополнительных мест хранения должны быть указаны приоритеты, например 1 и 2. Величина приоритета фактически влияет на порядок расположения колонок с дополнительными местами хранения в печатной форме «Бланк товарного наполнения». Чем выше приоритет, тем ближе к основному месту хранения будет показан альтернативный адрес хранения.

Места хранения номенклатуры можно заполнить не только из справочника "Склады", но и из справочника "Номенклатура". В справочнике "Номенклатура" места хранения номенклатуры могут быть заполнены непосредственно в карточке номенклатуры на закладке "Места хранения".
