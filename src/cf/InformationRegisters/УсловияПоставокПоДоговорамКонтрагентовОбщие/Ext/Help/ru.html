Регистр сведений "Условия поставок по договорам контрагентов (общие)"

Регистр предназначен для хранения данных документов "Условия поставок по договорам контрагентов", указанных в шапке документа.

На основании данных регистра производится анализ выполнения дополнительных условия по договорам контрагентов и строится отчет "Выполнение условий по договорам контрагентов".
