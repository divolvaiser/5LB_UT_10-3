Регистр сведений «Комплектующие номенклатуры»

В регистр сведений «Комплектующие номенклатуры» заносится информация о списке комплектующих для комплектов и наборов.

Для наборов используется отдельный вид номенклатуры - набор, который устанавливается при вводе набора в справочник «Номенклатура». Для наборов не ведется количественный и стоимостный учет, они используются только для быстрого подбора списка комплектующих, входящих в состав набора, в табличную часть документа. В качестве комплектующих в наборе могут быть указаны только товары, другие наборы и услуги не могут быть заданы в качестве комплектующих для набора. В состав набора могут входить товары с различными характеристиками, однако для самих наборов не ведется учет по характеристикам.

Комплектом может являться любой товар, для которого заполнен список комплектующих в регистре сведений «Комплектующие номенклатуры». Комплекты используются при оформлении операций комплектации (разукомплектации).

По комплектам ведется учет по характеристикам. В справочник"Номенклатура» может быть введен один комплект, имеющий различные характеристики. Для каждого комплекта с конкретной характеристикой может быть определен свой список комплектующих с определенными характеристиками.

Поле выбора «Характеристика комплекта» в регистре сведений позволяет производить отбор по составу комплектующих, заданных для комплекта с указанной характеристикой.  

Поле «Доля» в регистре сведений используется для определения доли стоимости комплектующего в общей стоимости комплекта при оформлении операции разукомплектации комплекта.
