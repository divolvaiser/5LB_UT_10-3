Регистр сведений «Комплектующие номенклатуры»

В этом регистре хранится список комплектующих, определенных для комплекта или набора в справочнике «Номенклатура».

Список комплектующих для набора используется для быстрого заполнения информации о составляющих набора в документ. Список комплектующих для комплекта используется при оформлении операций комплектации и разукомплектации. Комплектация и разукомплектация производится с помощью документа «Комплектация номенклатуры».
