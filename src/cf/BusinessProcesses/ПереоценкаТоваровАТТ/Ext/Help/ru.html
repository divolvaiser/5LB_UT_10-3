Бизнес-процесс «Переоценка товаров в АТТ»

Бизнес-процесс «Переоценка товаров в АТТ» необходим для проведения переоценки товаров на складе АТТ в строгой последовательности.

Процесс переоценки  состоит из трех этапов:

  • проведение инвентаризации товаров
  • установка новых цен
  • печать этикеток и ценников

Этапы процесса отражены на карте маршрута бизнес-процесса в виде точек «действия». Бизнес-процесс, следуя по маршруту, автоматически в каждой точке «действия» создает задачи. 

На первом этапе формируется задача на проведение инвентаризации на складе. На следующем шаге создается и проводится документ «Переоценка товаров в рознице» по тем позициям номенклатуры, цены на которые измелись. Исполнитель последнего этапа должен напечатать этикетки и ценники.  

Бизнес-процесс создается автоматически как этап процесса переоценки. Вариант прохождения бизнес-процесса по маршруту указывается в поле "Настройка"  из справочника "Настройки бизнес-процессов". В настройке для каждой точки «действия» можно указать представление, детальное описание, планируемый период выполнения и оповещения задачи. Исполнителем задач является пользователь, у которого в качестве «Физического лица» в справочнике «Пользователи» указан ответственное лицо склада. При необходимости некоторые точки можно отключить, что приведет к сокращению стандартной карты маршрута.

Для получения информации о текущей точке маршрута можно использовать «Карту маршрута», которая вызывается по кнопке  в командной панели формы.
