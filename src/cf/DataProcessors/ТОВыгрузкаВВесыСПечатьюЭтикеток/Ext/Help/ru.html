Обработка «Выгрузка данных в весы с печатью этикеток»

При помощи обработки «Выгрузка данных в весы с печатью этикеток» производится выгрузка информации о весовых товарах в весы с печатью этикеток, для проведения взвешивания и этикетирования весового товара. Работа с данной обработкой доступна только в том случае, если в списке торгового оборудования присутствуют физически подключенные весы с печатью этикеток.

Внимание. В весы с печатью этикеток выгружаются только те товары, у которых в позиции номенклатуры установлен признак «Весовой» и на закладке «Штрихкоды» определены коды весового товара.

Чтобы заполнить список загружаемых весовых товаров пользователю необходимо выполнить следующие действия:

  1. Указать модель весов, в которые нужно выгрузить товары.
  2. Установить тип цен, по которому должны быть установлены цены для выгружаемых товаров.
  3. Нажать кнопку «Заполнить» — список загрузки будет автоматически заполнен весовыми товарами.

Кнопка «Выгрузить в весы» предназначена для выгрузки сформированного списка в весы для проведения взвешивания и этикетирования товаров.

При выгрузке можно произвести отбор по группе номенклатуры или по конкретным номенклатурным позициям. Если необходимо выгрузить информацию только по тем товарам, которые имеются в наличии, необходимо установить  флажок «Только имеющиеся в наличии» и указать склад или список складов, на которых будет контролироваться остаток весовых товаров. Если склады не указаны, то контроль количества весового товара производится по всем складам.
