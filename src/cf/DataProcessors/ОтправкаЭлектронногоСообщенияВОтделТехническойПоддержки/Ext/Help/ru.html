Формирование обращения в отдел технической поддержки

Данный сервис предназначен для формирования обращения в отдел технической поддержки. В обращении обычно описывается возникшая проблема (ошибки выполнения, расчетов, представления информации, отсутствия / доступности данных или средств работы с данными).

Обращение представляет собой электронное сообщение, состоящее из заголовка (тема) и текста сообщения, к которому могут быть присоеденены файлы.

При открытии формы текст сообщения автоматически заполняется текстом, сформированным по данным параметров, указанных в форме "Настройка параметров обращения в отдел технической поддержки". Данный текст предназначен для облегчения понимания сотрудниками отдела технической поддержки описываемой ситуации. Этот текст может быть откорректирован. Никакой иной информации программой в обращение не добавляется.
Текст, вводимый пользователем, должен предшествовать данныму тесту.

Важно! В текст, сформированный по параметрам настройки, включаются только те параметры, информация о которых введена. 

В обращении рекомендуется подробно указать последовательность действий, приводящие к возникновению ошибочной ситуации. При описании необходимо использовать принятые термины и названия (форм, пунктов меню, закладок, элементов управления и т.д.) как они указаны в интерфейсе и документации.

Если по каким-либо причинам нужно заново сформировать текст, сформированный по параметрам настройки, то нажмите кнопку "Заполнить текст".
При этом весь введенный текст будет стерт и в тексте сообщения будет заново сформирован текст по параметрам настройки.

Если описание параметров настройки обращения устарело (например, был заменен компьютер (кроме локального), изменилось наименование организации, телефон, электронный адрес и т.д.), то для их редактирования нажмите кнопку "Настройка параметров" командной панели.
Некоторые параметры определяются и включаются программой в текст обращения автоматически. К таким параметрам относятся: описание конфигурации (версия, наименование, поставщик), описание компьютера, на котором выполняется формирование обращения, версия 1С:Предприятия 8.0.

Для лучшего понимания ситуации возможно присоединение файлов различных типов. Это могут быть файлы базы данных, файлы конфигураций, внешние обработки, документы с результатами отчетов (текстовые, табличные, HTML-документы), картинки, снятые с экрана, архивы и другая информация.

Для включения в состав обращения файлов перейдите на закладку "Вложения" и укажите их с помощью кнопок командной панели списка файлов.

Для отправки обращения нажмите кнопку "Отправить".

Сформированное письмо после отправки не сохраняется в базе данных. Сохранить письмо можно только в используемой почтовой программе, направив копию письма по адресу, указанному в настройках обращения в параметре "Отправитель: E-mail" раздела "Параметры отправки электронного сообщения". 
Для отправки копии письма по этому адресу в форме обращения следует установить флажок "Отправлять слепую копию себе".
