Обработка "Управление эквайринговой системой!

С помощью этой обработки осуществляется связь между программой "Управление торговлей" и эквайринговой системой.

При работе с эквайринговой системой можно выбрать следующие операции:

  1. Оплатить картой. При выполнении этой операции осуществляется связь с эквайринговой системой и фиксируется факт оплаты платежной картой (списание денежных средств с платежной карты).  
  2. Отменить платеж по карте - осуществляется связь с эквайринговой системой и отменяется факт оплаты платежной картой.
  3. Вернуть платеж по карте. Эта операция осуществляется в том случае, если покупатель вернул товар, который был оплачен платежной картой. При выполнении этой операции осуществляется связь с эквайринговой системой и выполняется операция перечисления денежных средств на платежную карту.  
  4. Выполнить сверку итогов по платежным картам - печать информации обо всех проведенных операциях по платежным картам.
