Обработка "Получение почты"

Обработка предназначена только для использования основного почтового клиента операционной системы.

Обработка предназначена для регистрации входящей почтовой корреспонденции пользователя с возможностью избирательной регистрации.

Пользователь получает список электронных писем, которые хранятся в папке «Inbox» почтовой программы, установленной на данном компьютере.

Можно настроить прием только непрочитанных электронных писем, а так же настроить показ только заголовков вложенных файлов.

При формировании списка писем, программа сама определяет те письма, которые уже зарегистрированы пользователем. Такие строчки выделяются серым цветом.

Для таких строчек кнопка командной панели «Открыть событие» становится доступна, и можно открыть форму документа «Событие», которым зарегистрировано данное электронное письмо.

У каждого письма в левой колонке можно проставить флажок – признак того, что письмо будет зарегистрировано, только тогда, когда указан отправитель письма и письмо еще не зарегистрировано в системе.

Отравитель при заполнении автоматически проставляется по адресу электронной почты которое ищется по заполненной контактной информации, если отправитель не найден, то пользователь может указать адрес электронной почты отправителя письма, для конкретного объекта системы открыв письмо двойным кликом мыши.

Письма регистрируются документами «Событие» с видом «Электронное письмо» и типом «Входящее». Вид объекта события определяется автоматически, в зависимости от типа отправителя.
