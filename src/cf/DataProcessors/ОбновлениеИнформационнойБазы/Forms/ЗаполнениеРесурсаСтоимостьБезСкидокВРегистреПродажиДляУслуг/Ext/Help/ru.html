Обработка «Заполнение ресурса "стоимость без скидок" в регистре "Продажи" для услуг»

Обработка используется в контексте обновления на ред. 10.3

Начиная с редакции 10.3 изменился способ заполнения ресурса Стоимость без скидок в регистре Продажи для услуг. Этот ресурс заполняется для Услуг также как и для Товаров стоимостью без скидки, в более ранних версиях этот ресурс для Услуг не заполнялся.

Для заполнения незаполненного ресурса, предназначена эта обработка.
