Обновление правил обмена с конфигурацией "Бухгалтерия предприятия"

Обработка используется в контексте обновления на релиз 10.3.4.

Предназначена для обновления правил обмена с конфигурацией "Бухгалтерия предприятия". Правила обмена в настройках обмена данными с конфигурацией "Бухгалтерия предприятия" будут заменены новыми правилами обмена.
