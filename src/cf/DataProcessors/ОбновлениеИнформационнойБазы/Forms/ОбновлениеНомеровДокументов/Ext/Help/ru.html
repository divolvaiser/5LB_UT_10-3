Обработка «Обновление номеров документов»

 

Обработка используется в контексте обновления на ред. 10.3

В ред. 10.3 изменился порядок префиксации документов: теперь префикс номера нового документа формируется в зависимости от узла РИБ, в котором создается документ.
Обработка «Обновление номеров документов» предназначена для корректной нумерации вновь создаваемых документов в том узле РИБ, который будет содержать документы, введенные ранее - до перехода на РИБ. При дальнейшем изложении такой узел называется «текущим».
Данная обработка запускается при соблюдении следующих условий:

  • после перехода на ред. 10.3 планируется начать работу в РИБ, причем ранее - до перехода на ред. 10.3 - РИБ не использовалась;
  • в информационной базе имеются документы, введенные в текущем периоде (году), которые теперь будут вводиться в разных узлах РИБ, и следовательно их номер должен учитывать префикс узла.

Данную обработку следует запускать только в «текущем» узле, и не следует (не имеет смысла) запускать в других узлах - тех, в которых ввод документов только начнется. 
Обработка должна запускаться в «текущем» узле после того, как:

  • будет назначен префикс «текущего» узла;
  • план обмена будет содержать хотя бы один подчиненный узел (это условие префиксации новых документов с учетом префикса РИБ).

В результате запуска обработки в  «текущем» узле будут созданы пустые не проведенные документы каждого вида с номерами, совпадающими с последними номерами документов, существующих в системе, и с префиксами, учитывающими префикс текущего узла РИБ.

Пример: Предположим последний документ  «Требование-накладная» в системе имеет номер «МСК00977». Решено начать работу в РИБ. Для текущего узла назначен префикс «ГУ». Если данную обработку не запускать, то при создании нового документа  «Требование-накладная» ему будет автоматически присвоен номер «ГУМСК000001» и нумерация будет нарушена. В результате запуска обработки будет создан пустой документ с номером «ГУМСК000977». После этого нумерация новых документов будет происходить корректно - очередной номер нового документа будет «ГУМСК000978».
